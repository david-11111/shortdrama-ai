# 修仙短剧第 1 集导演交接：废骨重生

## 用途

本文件交给 Agent 流水线使用。它不是新的剧本，而是导演约束：所有剧本、分镜、Seedream 参考图、Joy-Echo 视频提示词和剪辑判断，都必须服从这里的情感底层和质量标准。


## 总导演原则

这部短剧要反 AI 味。不要只堆修仙名词、金光、复仇口号和开挂爽点。观众要先相信这个人，再相信他的复仇。

第一集真正要拍的不是“废物逆袭”，而是“一个长期被忽视的人，第一次决定让别人听见自己说话”。

## 陆沉舟的情感底层

陆沉舟不是天生冷酷。他的克制来自长期的贫弱、退让和被忽视。

他不抢，不是高贵；他不争，不是洒脱；他沉默，不是没有话说。是因为他太早学会了：好东西轮不到自己，说了也没人听见。

所以第一集里，他的痛点不是被人当面辱骂，而是所有人习惯性越过他。测骨碑熄灭后，韩烈不用大骂，只要看都不看他，直接喊“下一个”，就够伤人。

这一世的重生，不是让他立刻无敌，而是让他第一次决定：我说的话，要有人听见。

## 表演指令

陆沉舟：

- 被越过时先低头，不急着瞪人。
- 想说话时有半拍停顿，话到嘴边先咽一下。
- 手会下意识缩进袖口，像怕别人看见旧伤。
- 拿起令牌时先擦掉灰尘，这是穷苦和自卑留下的习惯。
- 情绪不靠大喊，靠手、眼神、呼吸和沉默推进。
- 第一次反抗要像“终于决定为自己争一次”，不要像天生霸气外露。

沈青翎：

- 表面温和，实际是在替别人安排命运。
- 他可以说体面话，但不能真正尊重陆沉舟。
- 他的残忍来自“我觉得这样对你已经够好了”。

韩烈：

- 不需要一直骂人。
- 他最大的恶意是行政式忽视：划名册、丢令牌、喊下一个。
- 他把陆沉舟当成流程里的麻烦，不当成一个需要回应的人。

## 生活细节要求

每个关键镜头至少带一个人性细节，避免 AI 味空镜和大词堆砌。

可用细节：

- 令牌落地后沾了灰，陆沉舟先擦灰再握紧。
- 他想开口，但旁边弟子已经从他身边挤过去。
- 韩烈在名册上划掉他的名字，声音比辱骂更冷。
- 沈青翎给他“体面”，但不给他选择。
- 周围弟子不是集体大笑，而是短暂看一眼后继续自己的事。
- 陆沉舟听见沉剑渊剑鸣时，不喊复仇，只是眼神慢慢稳住。

禁用细节：

- 主角开场大吼复仇。
- 韩烈连续辱骂十几秒。
- 沈青翎邪笑或黑化表演。
- 所有人夸张围攻主角。
- 金色神光、系统面板、现代爽文梗。

## 剧情执行顺序

1. 读取生产包。
2. 先生成 R01-R06 参考图。
3. 检查陆沉舟是否有“克制、退让、想开口又收回”的气质。
4. 再生成 V01-V09 视频镜头。
5. 对 V03、V04、V07 做重点验收：是否拍出了“被忽视”。
6. 不合格镜头只重跑该镜头，不重跑全片。
7. ffmpeg 粗剪。
8. 加字幕、旁白和音效。

## 镜头重点

V03：

- 测骨碑熄灭。
- 韩烈不看陆沉舟，在名册划掉一笔。
- 令牌被随手丢到陆沉舟脚边。
- 韩烈喊“下一个”。
- 有人从陆沉舟身边挤过去。

V04：

- 陆沉舟手伸出想说话，又收回。
- 左手腕旧伤露出。
- 他低头捡起令牌，先擦灰。
- 不哭，不喊，不求饶。

V07：

- 韩烈终于注意到他还站着，伸手抽令牌。
- 陆沉舟小幅避开。
- 周围人第一次真正看向他。
- 陆沉舟说：“我还没测完。”

## Agent 输出验收

通过标准：

- 观众能感到陆沉舟不是冷酷模板男主，而是长期被忽视后开始争取自己位置的人。
- 第一集有生活细节，不只是修仙概念。
- 羞辱来自忽视和越过，不主要来自直接辱骂。
- 爽点来自“终于被看见”，不是无脑开挂。

失败标准：

- 主角像开场满级大佬。
- 镜头只剩炫光、喊话、打脸。
- 韩烈只会骂“废物”。
- 沈青翎像普通黑化反派。
- 画面漂亮但没有人性细节。
